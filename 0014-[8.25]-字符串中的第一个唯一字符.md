# 字符串中的第一个唯一字符
给定一个字符串，找到它的第一个不重复的字符，并返回它的索引。如果不存在，则返回 -1。

```
示例：

s = "leetcode"
返回 0
s = "loveleetcode"
返回 2
```

提示：你可以假定该字符串只包含小写字母。

## 利用find 函数
>`find()`从左到右寻找；`rfind()`从右到左寻找。如果该字符第一次出现的位置和最后一次出现的位置一样，就证明不重复。从前到后，遍历输出。
```C++
class Solution {
public:
    int firstUniqChar(string s) {
        for(int i=0;i<s.size();i++){
            if(s.find(s[i])==s.rfind(s[i]))
            return i;
        }
        return -1;
    }
};
```
